HackPiano

What is HackPiano?

The HackPiano is a MIDI-compatible piano made like a custom keyboard as my Hack Club Highway project.

Repo link: https://github.com/tinkering-townsperson/HackPiano

How did I make it?

Schematic/PCB

I used KiCad to make the schematic, and then assigned footprints to the symbols and imported the components onto the PCB. After moving them around a little, I routed the PCB.

Case

I Made the case in Autodesk Fusion. First I made my keyboard layout and exported it as a DHX file. I then imported it into Fusion.

Firmware

I wrote the firmware for this project in Python. I had to make a custom subclass that inherited from kmk.kmkkeyboard.KMKKeyboard and used the adafruit_midi library to send MIDI signals.
